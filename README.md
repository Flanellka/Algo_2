# mid7 Реализация хеш таблицы методом цепочек

### Описание
Нужно на C++ без использования std::unordered_table написать свою хеш-таблицу для хранения значений по ключу

+ Ключ: std::string (запрещено использовать std::hash)
+ Значение: int
+ Реализация хеш-функций: любой
+ Метод разрешения коллизий: любой
+ Реализация хеш-функций: любой

### Ограничения 
- Количество элементов : n <= 10^6
