# mid 1 

## Задача 
Реализовать многопоточный алгоритм перемножения матриц.

## Описание
Вам нужно написать обычное умножение матриц и параллельное. И сравнить время работы;

Размеры: 
- 2^6
- 2^8
- 2^10
- 2^12
  
Для каждого размера нужно произвести минимум 10 итераций И построить boxplot

(с числом потоков можете играться)
