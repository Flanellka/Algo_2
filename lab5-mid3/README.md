# mid3 Счетсчик свободных мест

## Описание

Есть [изображение](./cars.png). Там модель yolo нашла машины и свободные места

Но доступа к коду у вас нет, чтобы взять bboxes, по этому вам приходится работать с выходными изображениями алгоритма (картинка на выход)

Вам нужно написать алгоритм, который по фото посчитывал бы сколько свободных мест есть на парковке (считать зеленые прямоугольники)

*Будьте осторожны, некоторые полигоны являются не совсем прямоугольниками, по этому придется повозиться с координатами*


## Ограничение
*Писать код на python через numpy*

*UPD: запрещено использовать scipy*
